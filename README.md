# Clean-Robot

Le but de ce projet est de créer un robot capable de ramasser des déchets assez conséquents.  
Matériel utilisé:  
-deux moteurs cc  
-un module tank  
-une carte arduino  
-une carte L298N
-un bras robotique avec une pince au bout  
-une caisse pour stocker les dechets
